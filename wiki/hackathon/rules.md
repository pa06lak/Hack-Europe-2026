# Rules & Submission Requirements

> Source of truth: `[[hackathon-manual]]`. If anything here drifts from it, the manual wins.
> ⏱️ **Hard deadline 19:00.** Build the demo backward from submission — see `[[submission]]`.
> 🎯 We're on the **Attio — The Agentic CRM** track → `[[tracks-and-prizes]]`.

## Hard requirements to qualify (checklist)
- ⬜ Submit the project **by 19:00** (the cutoff for jury consideration)
- ⬜ Team of **max. 5 people** → `[[team]]`
- ⬜ Use **min. 3 partner technologies** → `[[partners]]`
- ⬜ Project **created newly at this hackathon** (boilerplates are allowed)

Miss any of these and we risk disqualification. Treat them as non-negotiable.

## What to submit
Two artefacts. Both required.

**1. Project presentation (video)**
- ⬜ **2-minute video demo** — Loom or equivalent
- Must include:
  - Detailed explanation of the solution
  - **Live walkthrough** of the key features
- Our script lives in `[[demo]]` — the one happy path: a buyer calls → qualified → matched listings → WhatsApp swipe link → swipe → write back, no agent in the loop.

**2. Open-source repository**
- ⬜ **Public GitHub repo** with the full source code
- Must include:
  - Comprehensive **README** with setup + install instructions
  - Clear docs of **all APIs, frameworks, and tools** used (for us: **SLNG, Attio, n8n, Gemini**; + Tavily if we wire the stretch)
  - Enough technical documentation for the jury to evaluate thoroughly

Full packaging checklist (video + repo + form) in `[[submission]]`.

## Competition mode
Two stages, ending in a live final. Details + how we score in `[[judging]]`.

**Stage 1 — Pre-Selection**
- Build anything aligned with your vision — total freedom of topic
- **5 finalist teams** advance: 4 Open Track Winners + 1 Track Winner → `[[tracks-and-prizes]]`
- Judged on **creativity** + **technical complexity**, with **bonus for effective use of partner tech**
- We're going for the **1 Track Winner** slot (Attio track) → `[[tracks-and-prizes]]`

**Stage 2 — Finalist Stage**
- All finalists present live to jury + audience
- **5-minute presentation** per team
- Jury picks the **top 3 winners**
- All **top 5** receive the Finalist Stage prizes → `[[tracks-and-prizes]]`
